# LFRic-Atmosphere-Training
A repository to host training for LFRic Atmosphere. Developed by the NGMS 'NG-UX' project and maintained by Momentum Partnership Team.

## Build the documentation

Building the documentation requires [Sphinx](https://www.sphinx-doc.org). On Met Office systems you can use `module load scitools` to add Sphinx to your enviroment. To build the documentation use:

```
git clone git@github.com:MetOffice/LFRic-Atmosphere-Training.git
cd LFRic-Atmosphere-Training
make html
```

You can then find the documentation in `build/html/index.html`.
