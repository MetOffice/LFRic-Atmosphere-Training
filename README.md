# LFRic-Atmosphere-Training
A repository to host training for LFRic Atmosphere. Developed by the NGMS 'NG-UX' project and maintained by Momentum Partnership Team.

## Build the documentation (at the Met Office)

module load scitools
make html

You can then find the documentation in `build/html/index.html`.

## Working Practises

When updating this training material, aim for clear and easy-to-understand revisions. Keep in mind that this repository may become public in the future. Therefore, contribute professionally and avoid including content that could be problematic for IT security or data protection. Utilise feature branches and introduce your changes through pull requests with descriptive names and detailed comments. Avoid committing minor updates directly to the main branch. If you have write access to the repository, please use branches instead of forks to enhance the visibility of improvements currently in development.

For reviews and to merge your changes into the main branch, please request assistance from [mo-JuliaCantarano](https://www.github.com/mo-JuliaCantarano), [menonarathy](https://www.github.com/menonarathy), or [BHFock](https://www.github.com/BHFock) from the Momentum Partnership Team. Feel free to log new ideas for the training in Issues or contact the partnership team directly.
