**************************************************
2. Unstructured Grid (UGRID) - LFRic output format
**************************************************

This module introduces the unstructured world of meshes which are now used in LFRic.

.. admonition:: Aims and objectives

   * Understand the concept of unstructured meshes and their advantages for atmospheric modelling.
   * Learn about the UGRID format and how it is used in LFRic.
   * Explore the structure of unstructured meshes and how they differ from structured meshes.
   * Gain familiarity with the tools and techniques for working with unstructured meshes in LFRic.
