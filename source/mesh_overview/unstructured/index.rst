******************
Unstructured mesh
******************

This chapter introduces unstructured meshes and the UGRID format,
which are essential for modern scientific data handling in fields
such as climate and weather modelling.
It explains the motivation for moving beyond traditional structured
grids, explores the advantages and complexities of unstructured data,
and describes how the LFRic model leverages these technologies for
greater flexibility, precision, and scalability.
