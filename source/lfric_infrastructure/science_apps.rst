LFRic Science Applications
==========================

The LFRic Atmosphere model is an `LFRic Science Applications <https://code.metoffice.gov.uk/trac/lfric_apps/browser/main/trunk/applications>`_ built on top of the LFRic Infrastructure. There are also other LFRic Science Applications built with the same infrastructure code, e.g. a linear model for data assimilation and an application for regridding and creating boundary conditions.
