4. Modelling with LFRic

This module covers how the Met Office's models are developed, configured and run within the Momentum® Framework.

   * To understand how Met Office models are developed
   * To be aware of working practices associated with model development
   * To become familiar with the MOSRS and wiki ticketing and documentation
   * To checkout, edit, and run a global model
   * To checkout, edit, and run a regional model
