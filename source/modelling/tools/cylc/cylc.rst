**********************
Cylc - workflow engine
**********************
* Cylc is a general-purpose workflow engine used to automate running
  groups of tasks in a specified order on computers.
* Cylc workflows can be organized running tasks at specific clock
  cycles.
* Cylc workflows represent the highest-level programming layer for
  organising the execution of our models, pre- and postprocessors.
