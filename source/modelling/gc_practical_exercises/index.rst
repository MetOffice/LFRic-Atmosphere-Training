**********************************
Exercises in Global Configurations
**********************************

The aim of this practical is to gain experience in editing, running, and analysing global configurations. You will:
.. admonition:: In this module you will learn about...

   * running a global configuration case
   * navigating the run directory and key output files
   * plotting model output fields for analysis
   * modifying and comparing experiment settings

.. admonition:: At the end of this module you should be able to...

   * edit and run a global configuration experiment
   * identify where important diagnostics and logs are stored
   * produce simple plots to inspect model behaviour
   * set up and evaluate experiment changes
